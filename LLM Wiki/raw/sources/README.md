# Raw Sources

把原始文章、访谈、播客笔记、会议纪要、网页摘录放在这里。

建议：

- 按学科再按年份分目录，例如 `raw/sources/math/2026/`。
- 尽量保留原文，不要把总结写进这里。
- 如果资料来自网页，可在文件头记录 URL、作者、日期。
- 图片和附件放到 `raw/assets/`，在 source 文件里引用相对路径。

当前推荐学科目录：

- `raw/sources/math/`: 数学
- `raw/sources/physics/`: 物理
- `raw/sources/tibetan-astronomy-calendar/`: 西藏天文历算
- `raw/sources/modern-astronomy/`: 现代天文学

如果未来增加新学科，优先使用稳定的英文 slug 作为目录名，再在 source frontmatter 中写中文标签。
