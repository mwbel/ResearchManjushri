# Skills

## Working Definition

- Skills 是一种面向 AI Agent 的能力封装机制：把完成一类任务所需的说明、步骤、脚本、模板、示例和资源放在一个可加载单元里。
- 当 Agent 识别到当前任务需要这类能力时，它可以加载对应 Skill，从而获得更具体的任务流程和工具使用方式。
- Skills 关注“如何稳定完成某类任务”；MCP 更偏向“如何以标准协议连接外部工具、数据和服务”。二者可以互补，但不是同一层东西。
- 在本知识库中，Skills 应作为“Agent 能力扩展机制”来理解，而不是某个具体产品或平台的名称。

## Why It Matters

- 它把通用 Agent 的能力从一次性提示词，推进到可复用、可维护的任务模块。
- 它让非模型层的知识、流程和工具调用方式能够沉淀下来，成为可迁移的操作能力。
- 它也解释了为什么同一个模型在不同工作流中表现差异很大：差异不只来自模型本身，也来自外部能力封装。
